file description:
dataprocessing.py: file for data processing, such as read data, sampling, training, create binary feature, find frequency words
nbc.py: file for running single nbc
test.py: run for different file and different size
HW3report.pdf: the report for project
plot.r: the code for plot result

command:
if you run single nbc, use nbc.py:
python nbc.py train-set.dat test-set.dat 7 1

if you want to run with different training size and different file:
python test.py
